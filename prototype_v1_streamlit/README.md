# V1 Streamlit Prototype

This folder contains the first working local Streamlit prototype.

It reads the provided Excel and JSON files directly with Pandas, computes recruitment KPIs and displays a first dashboard version.

This version is kept as a fallback and as a quick validation layer.

Run from the repository root:

streamlit run dashboard.py

Or run the archived copy:

streamlit run prototype_v1_streamlit/dashboard_v1.py
